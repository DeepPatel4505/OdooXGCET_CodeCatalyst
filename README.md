# Dayflow HRMS - Complete Human Resource Management System

A full-stack HRMS (Human Resource Management System) application built for Odoo X GCET, featuring a modern React frontend and a robust Node.js backend with PostgreSQL database.

## üåü Overview

Dayflow HRMS is a comprehensive solution for managing human resources operations including employee management, attendance tracking, leave management, payroll processing, and reporting. The system provides role-based access control for administrators, HR personnel, and employees.

## Tech Stack
Frontend: React, Vite, Tailwind CSS

Backend: Node.js, Express.js, Prisma ORM

Database: PostgreSQL
